convert_csv_to_lmm - CLI interface

Convert a CSV file of numerical values into a lmpy Matrix.

convert_csv_to_lmm [-h] [--header_rows HEADER_ROWS] [--header_cols HEADER_COLS]
                   in_csv_filename out_lmm_filename

convert_csv_to_lmm positional arguments

  • in_csv_filename - CSV filename to convert to lmm Matrix. (default: None)

  • out_lmm_filename - Filename for .lmm Matrix. (default: None)

convert_csv_to_lmm optional arguments

  • -h, --help - show this help message and exit

  • --header_rows HEADER_ROWS, -r HEADER_ROWS - The number of header rows in the CSV. (default: 0)

  • --header_cols HEADER_COLS, -c HEADER_COLS - The number of header columns in the CSV. (default: 0)
