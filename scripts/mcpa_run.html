mcpa_run - CLI interface

Compute a single run of MCPA metric.

mcpa_run [-h]
         pam_filename tree_matrix_filename env_matrix_filename biogeo_matrix_filename
         mcpa_matrix_filename f_matrix_filename

mcpa_run positional arguments

  • pam_filename - Path to PAM file. (default: None)

  • tree_matrix_filename - Path to encoded phylogeny. (default: None)

  • env_matrix_filename - Path to environment matrix. (default: None)

  • biogeo_matrix_filename - Path to biogeography matrix. (default: None)

  • mcpa_matrix_filename - Path to write computed MCPA values matrix. (default: None)

  • f_matrix_filename - Path to write F-Matrix. (default: None)

mcpa_run optional arguments

  • -h, --help - show this help message and exit
