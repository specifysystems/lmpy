Build Grid

build_ggrid - CLI interface

Build a grid shapefile that is used to define the sites of a PAM or other multivariate Matrix for lmpy operations.

build_ggrid [-h] [--config_file CONFIG_FILE]
            grid_filename min_x min_y max_x max_y cell_size epsg

build_ggrid positional arguments

  • grid_filename - File location to write grid shapefile. (default: None)

  • min_x - The minimum x value for the grid. (default: None)

  • min_y - The minimum y value for the grid. (default: None)

  • max_x - The maximum x value for the grid. (default: None)

  • max_y - The maximum y value for the grid. (default: None)

  • cell_size - The resolution of each grid cell (in EPSG map units). (default: None)

  • epsg - The EPSG code (map projection) for the grid. (default: None)

build_ggrid optional arguments

  • -h, --help - show this help message and exit

  • --config_file CONFIG_FILE - Path to configuration file. (default: None)
