==================
Lmpy Term Glossary
==================

.. glossary::

  DWCA
    A Darwin Core Archive is a packaged dataset of occurrence data.

  Data Wrangler
    Data wranglers are tools for filtering and / or modifying data using specified
    criteria or methodology.

  Matrix
    A Matrix, in the lmpy context, is an extension of numpy ndarray with metadata and
    headers.

  Point
    A Point object is a record of a specimen occurrence including metadata about the
    specimen and the spatial location where it was found.

  Grid
    A Grid is a set of gridded cells representing sites to be used for matrix
      creation.

  Species Distribution Model
    A species distribution model is a estimation of suitable and accessible areas for
      a particular species.

  Tree
    A Tree in lmpy represents a phylogenetic tree
