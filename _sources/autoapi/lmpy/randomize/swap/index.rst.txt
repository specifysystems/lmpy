.. function:: swap_randomize(matrix, num_swaps, max_tries=MAX_TRIES_WITHOUT_SWAP)

   Randomize a PAM matrix using the Gotelli swap method.

   :param matrix: A Lifemapper matrix object with binary elements
   :type matrix: Matrix
   :param num_swaps: The number of swaps to perform
   :type num_swaps: int
   :param max_tries: The maximum number of attempts to swap before failing.
   :type max_tries: int

   :returns: A randomized matrix.
   :rtype: Matrix

   :raises Exception: Raised when the maximum number of tries is reached without swapping.

   .. todo:: Allow num_swaps to be specified as a percentage of the matrix fill.
